# Unity 3D - SpaceShooter

This game is made based on Unity offical Spaceshooter tutorial with extension. Can be deployed to IOS and Android. Unity version: 2019.3.10

## What includes

- Player
- Camera and lighting
- Moving background
- Shooting action
- Hazards and Enemies
- Audio
- Joystick control
- Scoring
- Health bar
